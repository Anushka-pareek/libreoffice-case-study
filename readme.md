# LibreOffice Case Study CLI Tool
This project is a comprehensive Command Line Interface (CLI) application developed as an interactive accompaniment to the case study: *"The Fork in the Road: How LibreOffice Was Born from a Crisis of Trust"*. It follows all the important aspects of software development, such as JSON data parsing, command-line argument usage, and local file I/O for data persistence, all while strictly adhering to a non-GUI terminal environment constraint.

## Features
- **Full-Text Knowledge Search:** Search for concepts like open source governance, Oracle acquisition, and intellectual property ethics in a localized JSON database.
- **Chronological Timeline Generator:** Parse historical data to render a formatted timeline of the OpenOffice.org to LibreOffice fork.
- **Interactive Terminal Quiz:** A multi-question quiz that tests your knowledge of the case study.
- **Data Persistence:** Automatically keeps track of your performance and saves it to a local `user_stats.json` file, allowing you to view your historical performance.

## Prerequisites
- **Python 3.6 or higher** installed on your system.
- Standard terminal environment (Linux bash, macOS Terminal, Windows CMD, or PowerShell).
